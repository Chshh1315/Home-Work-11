Список самых необходимых комманд git в консоли

  • git init - инициализация гит репозитория
  • git status - список измененных файлов
  • git add (имя файла) - добавить файл в репозиторий
  • git add . - добавить все файлы в репозиторий
  • git commit -m "коментарий" - добавить коммит с комментарием
  • git log - список всех коммитов
  • git checkout (хэш коммита или название ветки) - передвижение по коммитам или веткам
  • git checkout -b "название ветки" - создает новую ветку из текущей
  • git branch - список всех веток
  • git merge (название ветки) - объединить ветки, указываем название той ветки которую хотим объединить с текущей
  • git remote add origin (ссылка) - используется для подключении к удаленному репозиторию
  • git branch -m master - задаем главную ветку
  • git push -u origin master - оставляем название ветки master, а не main
  • git diff - показвает что было изменено от предедущего коммита к текущему
  • git push origin (название ветки) - добавить в репозиторий ветку
  • git pull origin (название ветки) - загрузить ветку с репозитория
